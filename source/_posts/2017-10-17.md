# 纯css实现 实心箭头
-----------------
```html
//html
    <div class="content">
        <span class="top"></span>
        <span class="right"></span>
        <span class="bottom"></span>
        <span class="left"></span>
    </div>
```
```css
/* css */
    .top ,.right,.bottom,.left{
        float: left;
        clear: both;
        margin-bottom: 23px;
    }
    /* 向上 */
    .top {
        font-size: 0;
        line-height: 0;
        border-width: 10px;
        border-color: #000;
        border-top-width: 0;
        border-style: dashed;
        border-bottom-style: solid;
        border-left-color: transparent;
        border-right-color: transparent;
    }
    /* 向右 */
    .right {
        font-size: 0;
        line-height: 0;
        border-width: 10px;
        border-color: #000;
        border-right-width: 0;
        border-style: dashed;
        border-left-style: solid;
        border-top-color: transparent;
        border-bottom-color: transparent;
    }
    /* 向下 */
    .bottom {
        font-size: 0;
        line-height: 0;
        border-width: 10px;
        border-color: #000;
        border-bottom-width: 0;
        border-style: dashed;
        border-top-style: solid;
        border-left-color: transparent;
        border-right-color: transparent;
    }
    /* 向左 */
    .left {
        font-size: 0;
        line-height: 0;
        border-width: 10px;
        border-color: #000;
        border-left-width: 0;
        border-style: dashed;
        border-right-style: solid;
        border-top-color: transparent;
        border-bottom-color: transparent;
    }
```

# 纯css实现 方向箭头
-----------------
```html
    <div class="arrow-bottom"></div>
```
```css
    .arrow-bottom {
        width: 20px;
        height: 4px;
        margin: 0 auto 7px;
        border-left: 4px solid transparent;
        border-right: 4px solid transparent;
        border-bottom: 4px solid #000;
        transform: rotate(45deg);
        transform-origin: left;
    }

    .arrow-bottom:after {
        content: '';
        display: block;
        width: 100%;
        height: 100%;
        border-left: 4px solid transparent;
        border-right: 4px solid transparent;
        border-top: 4px solid #000;
        position: absolute;
        right: -10px;
        top: -14px;
        transform: rotate(90deg);
        transform-origin: bottom;
    }
```
效果：![mark](http://oopl89lfl.bkt.clouddn.com/myerlee/20171017/194444418.png?imageslim)

其他方向只需要改变`transform`即可

# 纯css实现 带箭头对话框
-----------------
```html
    <div id="dialog"></div>
```
```css
    .dialog {
        width: 100px;
        height: 100px;
        background-color: #ccc;
        position: relative;
        border: 4px solid #333;
    }

    .dialog:after,
    .dialog:before {
        border: solid transparent;
        content: ' ';
        height: 0;
        left: 100%;
        position: absolute;
        width: 0;
    }

    .dialog:after {
        border-width: 9px;
        border-left-color: #ccc;
        top: 15px;
    }

    .dialog:before {
        border-width: 14px;
        border-left-color: #333;
        top: 10px;
    }

```
